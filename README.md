&gt; 一款真正懂你的漏洞打点工具 —— 不只是扫描，更是你的 AI 安全搭档

## 一句话介绍

Yscan 是面向实战的**轻量化可视化漏洞打点平台**，集成 AI 智能分析、流量威胁研判、空间测绘与 POC 全生命周期管理，让资产发现到漏洞验证的每一步都更简单、更智能。

## 核心能力

### 🤖 AI 安全助手 —— 从"问答"到"执行"

不只是聊天机器人，而是能**动手干活**的安全工作台：

- **任务编排** —— 一句话创建扫描任务、查询进度、汇总报告
- **结果解读** —— 自动分析端口、指纹、漏洞、弱口令，标红高危风险
- **链路扫描** —— 智能分阶段：Web 摸底 → 端口识别 → 漏洞验证
- **文档理解** —— 上传 DOCX/TXT，AI 结合材料给出针对性结论
- **多会话管理** —— 复杂分析不断线，支持置顶、历史回溯

&gt; 安全小白？问它就行。资深选手？让它帮你省掉重复劳动。

---

### 🌐 综合测绘 —— 多引擎，一个入口

- 各引擎独立配置账号和检索策略
- 统一字段聚合：URL、IP、端口、组件、地域
- 支持当前结果 / 选中标签 / 全量数据多种导出方式

---

### 📊 流量分析 —— 从"看见"到"看清"

解决安全团队三大痛点：**看不清、判不准、处置慢**

| 能力 | 说明 |
|-----|------|
| 接入 | 实时流量采集 / PCAP 批量导入 |
| 检索 | 事件ID、源目标IP、风险等级、处置状态多维筛选 |
| 聚合 | 按攻击源聚合告警，优先展示高风险链路 |
| 可视化 | 感染源排行、受害目标、事件分布一目了然 |

---

### 🎯 POC 管理 —— 全流程自动化

**添加 POC → 自动生成指纹 → 自动关联工作流 → 直接开扫**

- **AI POC 生成器** —— 描述漏洞，AI 帮你写 POC
- **智能匹配** —— 拷贝 Nuclei POC 到目录，自动匹配指纹入队
- **可视化编排** —— Workflow、指纹管理图形化操作
- **灵活验证** —— 批量扫描 or 单点精准打击，随你选
